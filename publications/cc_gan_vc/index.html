Audio samples of "Non-Parallel Many-To-Many Voice Conversion Using Cross-Cycle Consistency Loss"

Abstract: In this paper, we present a disentangled adversarial network for many-to-many voice conversion using non-parallel data. In our proposed method, content feature and attribute feature are ex- tracted from inputs with different speaker domains. The content feature is obtained from a content adversarial network while the attribute feature is represented by a style embedding. Another adversarial network serves a generator which synthesizes output voice based on content feature and attribute feature. A cross- cycle consistency loss is used to make content and attribute mi- gration possible between a non-parallel voice pair. Experiments are conducted and subjective evaluation shows that our model can obtain good voice quality.
